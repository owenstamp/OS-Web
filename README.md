# Owen's Developer Portfolio

A modern, responsive static HTML portfolio site for showcasing personal projects, skills, and contact information.

## Features
- Responsive design
- About, Projects, Skills, and Contact sections
- Simple contact form (placeholder)

## Usage
Open `index.html` in your browser to view the site.

## Customization
- Update project details in the Projects section of `index.html`.
- Add or modify skills in the Skills section.
- Replace placeholder contact form logic in `script.js` with your own backend integration if needed.
